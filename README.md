## Features

- AES-256-GCM symmetric encryption
- RSA (2048, 3072, 4096, 8192 bit)
- EC Elliptic Curve (P-256, P-384, P-521)
- Ed25519 asymmetric encryption
- PBKDF2 key derivation (600,000 iterations)
- Password strength indicator
- File encryption/decryption
- Modern dark theme GUI
- Multi-threaded operations
- Key export to PEM format
- Progress bar for file operations

## Installation

Prerequisites: Python 3.8 or higher

Install dependencies:
```bash

```
or
```bash

```

Run the application:
python main.py

## How to Use

### AES-256 Tab

Setting up a key:
- Click Generate to create a random 256-bit key
- Or type any password (strengthened with PBKDF2)
- Click Paste to paste a key from clipboard
- Click the eye icon to show or hide password

Encrypting text:
1. Enter your key
2. Type text in the Input field
3. Click Encrypt
4. Encrypted text appears in Output field

Decrypting text:
1. Ensure the correct key is entered
2. Paste encrypted text in the Input field
3. Click Decrypt

File encryption:
1. Set your key
2. Click Encrypt File and select a file
3. Choose save location
4. For decryption, click Decrypt File

Quick tips:
- Use Swap button to switch Input and Output
- Password strength indicator shows weak, medium, or strong
- For hex keys, enter exactly 64 characters

### Asymmetric Tab

Generating keys:
1. Select key type: RSA, EC, or Ed25519
2. Choose key size or curve
3. Click Generate Keys

Saving keys:
1. Click Save Keys
2. Choose a folder
3. Two files created: private_key.pem and public_key.pem
4. Keep private key secure

Importing keys:
1. Click Import Key
2. Select PEM file
3. Public key is auto-derived

Encrypting:
1. Load public key
2. Type message in Input
3. Click Encrypt

Decrypting:
1. Load private key
2. Paste encrypted text in Input
3. Click Decrypt

File operations work the same as AES tab.

## Export to EXE

Install PyInstaller:
pip install pyinstaller

Basic build:
```bash
pyinstaller --onefile --windowed --icon=icon.ico --add-data "icon.png;." --name "AlienEncryption" main.py
```

With custom icon:
pyinstaller --onefile --windowed --icon=icon.ico --name AlienEncryption main.py

Executable will be in the dist folder.

To reduce file size, use virtual environment:
python -m venv build_env
build_env\Scripts\activate
pip install PyQt6 cryptography pyinstaller
pyinstaller --onefile --windowed --clean --name AlienEncryption main.py

## Security Notes

- AES keys should be 32 random bytes
- Weak passwords strengthened with PBKDF2
- RSA 4096-bit or Ed25519 for long-term security
- Never share private keys
- This software has not been professionally audited
